Árbol Binario de Búsqueda (ABB) - Documentación de Ejecución y Prueba

	Este programa implementa un Árbol Binario de Búsqueda (ABB) en Python, 
	proporcionando una interfaz de línea de comandos (CLI) para que los usuarios puedan crear, 
	manipular y visualizar el árbol. El programa permite insertar, buscar, eliminar, 
	cargar datos desde un archivo y visualizar el árbol utilizando Graphviz.

Uso del programa

El programa presenta un menú interactivo que permite al usuario realizar diversas operaciones en el Árbol Binario de Búsqueda:

	Insertar: Permite al usuario insertar un número en el árbol.
	Buscar: Permite al usuario buscar un número en el árbol.
	Eliminar: Permite al usuario eliminar un número del árbol.
	Cargar desde archivo: Permite al usuario cargar números desde un archivo .txt y construir el árbol.
	Visualizar con Graphviz: Convierte el árbol a un formato visual utilizando Graphviz y muestra la representación visual del árbol.

Prueba del programa

	Para probar el programa, puede seguir estos pasos:

	Inserte algunos números en el árbol utilizando la opción de inserción.
	Pruebe la opción de búsqueda para verificar si los números insertados están presentes en el árbol.
	Elimine algunos números utilizando la opción de eliminación y verifique que se eliminen correctamente.
	Cargue datos desde un archivo .txt utilizando la opción de carga desde archivo y observe cómo se construye el árbol.
	Utilice la opción de visualización con Graphviz para ver la representación visual del árbol y verifique que coincida con la 	estructura esperada.
