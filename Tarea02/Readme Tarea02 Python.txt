Recursividad 

Este programa proporciona varias funciones implementadas utilizando recursión. Puedes utilizar estas funciones para realizar diversas operaciones matemáticas, como convertir números a binario, contar dígitos, calcular la raíz cuadrada entera, convertir números romanos a decimales y sumar números enteros hasta un valor dado.

Funcionalidades Implementadas

	Convertir un número a binario: Convierte un número entero en su representación binaria.
	Contar dígitos de un número: Cuenta la cantidad de dígitos en un número entero.
	Calcular raíz cuadrada entera: Calcula la raíz cuadrada entera de un número entero.
	Convertir número romano a decimal: Convierte un número romano en su equivalente decimal.
	Sumar números enteros hasta un valor: Calcula la suma de todos los números enteros desde 0 hasta un valor dado.

Requisitos
	Python 3.x

Instrucciones de Uso

Descarga o clona este repositorio en tu máquina local.
Abre una terminal y navega hasta el directorio donde se encuentra el archivo tarea02.py
Ejecuta el programa escribiendo python tarea02.py en la terminal.
Se mostrará un menú con las diferentes opciones. 
Selecciona la opción deseada e ingresa los valores cuando se te soliciten.
El programa realizará la operación correspondiente y mostrará el resultado.
Puedes repetir el proceso para utilizar otras funcionalidades o salir del programa seleccionando la opción correspondiente en el menú.

Ejemplo:

Bienvenido al menú de RECURSIVIDAD, escoja un método por favor:

1. Convertir un número a binario
2. Contar dígitos de un número
3. Calcular raíz cuadrada entera
4. Convertir número romano a decimal
5. Sumar números enteros hasta un valor
6. Salir

Ingrese el número de la opción que desea ejecutar: 1

Ingrese un número entero para volverlo binario: 10
El número 10 en binario es:
1010

Presione Enter para volver al menú...
